Aplicativo de consulta motos

Consulta motos es una aplicativo realizado en React el cual consume una Api de un aplicativo de generacion de formularios masivos llamado KoboToolBox, este aplicativo permite a los usuarios verificar el estado de registro de su motocicleta en la base de datos del distrito de cartagena de indias.
